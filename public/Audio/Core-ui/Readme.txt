# 💎 دليل النظام الصوتي للواجهات الأساسية (Core UI Audio System)

مرحباً 👋

هذا المجلد يحتوي على **"القلب النابض"** للنظام الصوتي الداخلي لمنصة "المعلم الذكي". 
بدلاً من تشتيت الأصوات في كل صفحة، تم بناء هذا النظام (Audio Design System) ليكون مركزياً، حيث تم دمج كافة مؤثرات الواجهة (UI Sounds) في ملف **Audio Sprite** واحد. 

هذا النهج يضمن أقصى أداء (Zero Latency)، يمنع إعادة التحميل عند التنقل بين الصفحات، ويقلل طلبات الخادم (HTTP Requests) بشكل جذري.

## 📁 المحتويات:
* **ملفات الصوت المدمجة:** `core-ui-sprite.webm`، `core-ui-sprite.m4a`، `core-ui-sprite.mp3`.
* **خريطة التوقيتات:** `core-ui-sprite.json`.

---

## ⚙️ طريقة الاستدعاء (Howler.js):

يجب تهيئة هذا الكائن الصوتي **مرة واحدة فقط** (Global State / Layout) عند دخول الطالب للمنصة، ليكون متاحاً للاستدعاء من أي واجهة (Component).

```javascript
import { Howl } from 'howler';
import coreUiSpriteData from './core-ui-sprite.json';

// تهيئة الصوت العام للمنصة
export const coreSounds = new Howl(coreUiSpriteData);

// مثال للاستخدام عند النقر على أي بطاقة:
// coreSounds.play('card-feature-click');
```

---

## 🎛️ قائمة التفاعلات المتاحة (Triggers List):

استخدم هذه الأسماء (Keys) بداخل دالة `play()` لتشغيل الصوت المناسب، وفقاً للحالات التالية:

### 1. النظام والتنقل (Navigation & System)
* **`ui-menu-toggle`** : عند فتح/إغلاق القائمة الجانبية (Sidebar).
* **`ui-nav-click`** : عند التنقل بين عناصر القائمة الجانبية.
* **`ui-theme-toggle`** : عند تبديل المظهر (ليلي/نهاري) ☀️🌙.
* **`ui-action-locked`** : النقر على ميزة "قريباً" أو أي زر مقفل 🔒.
* **`ui-nav-premium`** : النقر على عناصر الباقة المميزة أو النوافذ المنبثقة للترقية ✨.
* **`ui-logout`** : عند الضغط على زر تسجيل الخروج ⬅️.
* **`ui-error-msg`** : عند ظهور أي تنبيه أو رسالة خطأ عامة في المنصة ⚠️.

### 2. التفاعل والأزرار (Interactions)
* **`ui-text-link`** : مخصص لجميع الروابط النصية، زر النسخ، أيقونات القراءة، وزر الإلغاء 🔗.
* **`card-feature-click`** : عند اختيار أو النقر على البطاقات الكبيرة (مثل: اختيار مادة، شخصية معلم، نوع الاختبار).
* **`btn-primary-action`** : للأزرار البنفسجية الأساسية (حفظ التغييرات، التالي، تسليم الاختبار).
* **`btn-secondary-action`** : للأزرار الرمادية الثانوية أو أزرار النوافذ المنبثقة.
* **`btn-hero-start`** : زر الإجراء الرئيسي في الصفحة الرئيسية (ابدأ الرحلة) 🌟.
* **`ui-dropdown-open`** : عند فتح القوائم المنسدلة (اختيار الصف) أو الأكورديون (توسيع مراجعة الأسئلة).
* **`ui-checkbox-check`** : عند اختيار إجابة من الخيارات (A, B, C, D) في الاختبار ✅.

### 3. الدردشة مع المعلم الذكي (Chat)
* **`chat-msg-send`** : عند إرسال الطالب للسؤال أو الرسالة ✈️.
* **`chat-msg-receive`** : عند ظهور/اكتمال رد المعلم الذكي في الشاشة.

### 4. الاختبارات والنتائج (Gamification)
* **`quiz-loader-done`** : عند اكتمال شريط التحميل الخاص بـ "المعلم يقوم بتجهيز الاختبار" لـ 100%.
* **`quiz-result-perfect`** : لشاشة النتيجة النهائية الممتازة (100%) - بطل! 🥇
* **`quiz-result-good`** : لشاشة النتيجة العالية والمتوسطة (60% إلى 90%) - رائع/جيد 📈
* **`quiz-result-retry`** : لشاشة النتيجة الضعيفة (30% وأقل) - لنحاول مرة أخرى 💔
